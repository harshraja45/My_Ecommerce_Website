# 🛒 E-Commerce Website (React + Redux)

A modern e-commerce web application built using React and Redux. This project demonstrates state management, reusable components, and a clean UI for online shopping.

---

## 🌐 Live Demo

👉 [View Live Project](https://my-ecommerce-website-wheat-rho.vercel.app)

---

## 🚀 Features

* 🛍️ Browse products
* ➕ Add to cart
* ❌ Remove items from cart
* 🔄 Real-time cart updates using Redux
* 📱 Responsive design

---

## 🛠️ Tech Stack

* React (Frontend UI)
* Redux (State Management)
* JavaScript (ES6+)
* CSS / Tailwind / Bootstrap (if used)
